# E-Library Frontend

## Setup

```sh
npm install
npm run dev
```

## Features
- View book list
- Search for books
- Read books using PDF.js

## API Endpoints Used
- `GET /books` - Fetch all books
- `GET /books/:id` - Fetch book details
- `POST /books` - Add a book
- `PUT /books/:id` - Update book details
- `DELETE /books/:id` - Delete a book
